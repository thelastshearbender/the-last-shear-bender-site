# The Last Shear Bender — Booking Site

💈 Premium barber booking page for Ottawa.

## 🚀 DEPLOY IN 3 MINUTES (Netlify Drop — FREE)

### Step 1: Unzip this folder
You should see `index.html` and `netlify.toml`.

### Step 2: Go to Netlify Drop
Open: **https://app.netlify.com/drop**

(No account needed to start — you'll create one after deploy to save the site.)

### Step 3: Drag & Drop
Drag the **entire unzipped folder** onto the Netlify Drop page.

Wait 30 seconds. Done! ✅

You'll get a random URL like `dazzling-pie-a1b2c3.netlify.app`.

### Step 4: Rename the URL
1. Click "Site settings" → "Change site name"
2. Change to: **thelastshearbender**
3. Your new URL: **https://thelastshearbender.netlify.app** 🎉

### Step 5: Share It!
Add to your Instagram bio:
```
💈 Ottawa's Premium Barber
📲 Book: thelastshearbender.netlify.app
```

## 📱 How It Works

**For clients:**
- Visit your link → Pick service → Pick date/time → Enter name & phone → DONE
- Takes under 60 seconds

**For you (owner access):**
- Visit your site and tap the "⚙️ OWNER" button (top-right)
- See all bookings, revenue, and this week's count
- Delete bookings if needed

**Important:** Bookings are saved in the browser (localStorage). They're visible on whichever device the client used.

## 🔄 To Update Content Later

1. Edit `index.html`
2. Find the `BRAND` or `SERVICES` section at the top of the `<script>` block
3. Change values, save, re-upload to Netlify (drag & drop again onto your existing site dashboard)
